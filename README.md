# common-export

1. Support exporting CSV and Excel files
2. Support data acquisition method export and memory data export

## How to use it
```xml
<dependency>
	<groupId>com.github.catdou</groupId>
	<artifactId>common-export</artifactId>
	<version>1.2</version>
</dependency>
```
